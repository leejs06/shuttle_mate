# 셔틀메이트(ShuttleMate) - 기능 메뉴얼 - (2026-03-30)

1. 랜덤매칭
  ㄴ 랜덤 매칭은 4명의 플레이어들의 이름을 재각각 입력받아 2vs2 대진을 짜주는 랜덤 매칭 기능입니다.

2. 모임 만들기
  ㄴ 모임 만들기는 로그인한 사용자만 이용이 가능하며, 본인이 속한 모임을 만들어서, 모임 내에 회원들을 다 등록 후 랜덤매칭, 수동매칭 기능 사용이 가능합니다.

  2.1 모임 만들기 > 랜덤매칭
    ㄴ 위 경로에 랜덤매칭은 1번 내용과 다르게 더 세밀하고, 정밀한 밸런스 조정 로직이 적용되어, 최대한의 밸런스가 되는 매칭을 잡아주는 기능입니다.
      ㄴ 이후 승/패를 선택 시 기록까지 되며, 매칭된 회원들이 이 날에 몇 경기를 하였는 지 체크도 가능합니다. - 게임의 공정성을 위한 기능
    
  2.2 모임 만들기 > 수동매칭
    ㄴ 위 경로에 수동매칭은 회원들의 정보가 들어간 리스트가 UI에 띄워져서, 직접 경기를 사람이 수동으로 매칭 시키는 기능입니다.
      ㄴ 또한 수동매칭도 승/패를 선택하여, 기록까지 할 수 있고, 몇 경기를 하였는 지 바로 알 수 있게 UI에 띄울 생각입니다. - 이것 또한 공정성을 위해 필요한 기능

3. 회원가입 / 로그인

  3.1 회원가입
    ㄴ 회원가입을 하면, 회원가입한 회원의 ID, PW, 성별, 연령, 급수 정보가 저장됩니다.
  3.2 로그인 
    ㄴ 로그인을 하면, 모임 만들기 기능을 사용할 수 있으며 왠만한 모든 기능 사용이 가능합니다.
